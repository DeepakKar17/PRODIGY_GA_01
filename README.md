# 🎓 Fine-Tuning GPT-2 on GCEK College Data

This project focuses on fine-tuning the GPT-2 language model using custom data related to GCEK (Government College of Engineering, Kannur). The model is trained to generate context-aware responses based on college-specific information.

---

## 🚀 Project Overview

* 🔍 Fine-tuned **GPT-2** model on custom dataset
* 🏫 Dataset based on GCEK college-related information
* ☁️ Training performed using Google Colab
* 🤖 Goal: Generate meaningful and context-aware text about GCEK

---

## 📂 Project Workflow

1. Data Collection & Cleaning
2. Tokenization of text data
3. Fine-tuning GPT-2 model
4. Evaluation of generated outputs
